欢迎使用艾尔登法环练习工具!

要运行工具，只需要在游戏启动后双击`jdsd_er_practice_tool.exe`挂载。
要使用no-logo mod，复制`dinput8.dll`到`eldenring.exe`所在的游戏目录.

请使用添加`steam_appid.txt`绕过EAC启动的方式。
